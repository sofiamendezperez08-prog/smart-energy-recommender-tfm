# Smart Energy Recommender

Repositorio asociado al Trabajo Fin de Máster:

**Desarrollo de un sistema inteligente de recomendación energética residencial mediante clustering
de perfiles de consumo y optimización multicriterio sobre tarifas PVPC**

## Objetivo

Desarrollar un sistema inteligente capaz de recomendar las mejores horas para utilizar dispositivos domésticos aprovechando los precios dinámicos del PVPC.
