# Smart Energy Recommender

Repositorio asociado al Trabajo Fin de Máster:

**Desarrollo de un sistema inteligente de recomendación energética residencial mediante clustering
de perfiles de consumo y optimización multicriterio sobre tarifas PVPC**

## Objetivo

Desarrollar un sistema inteligente capaz de recomendar las mejores horas para utilizar dispositivos domésticos aprovechando los precios dinámicos del PVPC.

## Contenido/Notebooks

- eda_dataset_uci.ipynb
- pvpc_prices.ipynb
- user_profiles_clustering.ipynb
- smart_energy_recommender.ipynb
- smart_energy_recommender_v2.ipynb
- smart_energy_recommender_v3.ipynb

## Tecnologías

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- API e·sios (Red Eléctrica de España)

## Datos

Se emplean:

- Individual Household Electric Power Consumption Dataset (UCI)
- Precios PVPC obtenidos mediante la API oficial de e·sios
- Intensidad horaria de emisiones de CO₂
